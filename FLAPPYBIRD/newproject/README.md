GameProject

A simple Flappy Bird game developed in Java using Java Swing and Object-Oriented Programming (OOP) concepts.

Features
Bird movement with gravity
Random pipe generation
Collision detection
Score tracking system
Restart game functionality
User-friendly graphical interface
OOP Concepts Used
Classes and Objects
Encapsulation
Constructors
Object Interaction
Event Handling

How to Play
Press the Spacebar to make the bird jump.
Avoid colliding with the pipes.
Fly through the gaps between the pipes.
Earn points by successfully passing pipe sets.
The game ends if the bird hits a pipe or falls off the screen.
Press Spacebar after Game Over to restart the game.
How to Run

Compile all Java files:

javac *.java

Run the application:

java App
